# 目录结构

根据 `README.md`，本项目为 C# 教学解决方案。

```text
CSharpTeachingSolution/
├── CSharpTeachingSolution.sln
├── MainApp/                # 主入口控制台应用
├── MainWeb/                # Web 展示模块
├── Module.*/               # 具体的学习模块
├── Web.*/                  # Web 游戏/应用模块
├── docs/                   # 项目说明文档
└── docker/                 # Docker 部署文件配置
```
